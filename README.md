## Setup Instructions

### Backend Setup

1. Navigate to the backend directory:
```bash
cd backend
```

2. Install dependencies:
```bash
npm install
```

3. Start the backend server:
```bash
npm start
```

The backend will run on `http://localhost:3001`

### Admin Dashboard Setup

1. Navigate to the admin directory:
```bash
cd admin
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The admin dashboard will be available at `http://localhost:5173` (or the port shown in terminal)

## API Endpoints

- `GET /api/leads` - Get all leads (supports `?date=YYYY-MM-DD` and `?employeeId=X` query params)
- `GET /api/leads/by-date` - Get leads grouped by date
- `GET /api/leads/by-employee-date` - Get leads grouped by employee and date
- `GET /api/employees` - Get all employees
- `GET /api/stats` - Get statistics (total leads, leads by employee, leads by date)
- `POST /api/leads` - Create a new lead

## Usage

1. Start the backend server first
2. Start the admin dashboard
3. View leads organized by date
4. Use filters to view specific dates or employees
5. Switch between grouped and list view modes
